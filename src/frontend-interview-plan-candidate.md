### 📋 Задание 1. Каррирование (JavaScript)

**Сложность:** 🟡 | ⏱ ~15 мин

Реализуйте функцию `curry(fn)`, которая превращает обычную функцию в каррированную. Функция должна принимать аргументы по одному или группами, пока не соберёт все.

**Сигнатура и тесты:**

```js
const sum = (a, b, c) => a + b + c;
const curriedSum = curry(sum);

console.log(curriedSum(1)(2)(3));   // 6
console.log(curriedSum(1, 2)(3));   // 6
console.log(curriedSum(1)(2, 3));   // 6
console.log(curriedSum(1, 2, 3));   // 6
```

**Дополнительно:** подумайте, как поведёт себя решение с функцией, имеющей значения по умолчанию или rest-параметры.

### 📋 Задание 5. Типобезопасный Event Emitter (TypeScript)

**Сложность:** 🟡 | ⏱ ~25 мин

Реализуйте фабрику `createEmitter<E>()`, создающую эмиттер событий с типобезопасными подписками. Имена событий и типы payload задаются интерфейсом `E`.

**Заготовка:**

```ts
// TODO: реализовать createEmitter и нужные интерфейсы

interface AppEvents {
  login: { userId: string; token: string };
  logout: void;
  error: { code: number; message: string };
}

const emitter = createEmitter<AppEvents>();

emitter.on('login', (payload) => {
  // payload должен иметь тип { userId: string; token: string }
  console.log(payload.userId);
});

emitter.emit('login', { userId: '1', token: 'abc' }); // OK
emitter.emit('logout');                                // OK, payload не нужен
// emitter.emit('login', { userId: '1' });             // должно быть ошибкой: нет token
// emitter.emit('unknown', {});                        // должно быть ошибкой: нет такого события

const off = emitter.on('error', (e) => console.log(e.code));
off(); // отписка
```

**Дополнительно (опционально):** метод `once()` для однократной подписки, `offAll()` для очистки.
