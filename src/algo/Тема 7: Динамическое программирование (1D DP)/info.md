# Тема 7: Динамическое программирование (1D DP)

> **Главная идея темы:** оптимальное решение задачи выражается через оптимальные решения подзадач. Состояние — одно измерение (индекс / сумма / длина). Сначала рекуррент, потом оптимизация памяти.

---

## Когда применять

1. Задача просит **оптимум** (min/max/count) или «можно ли».
2. Есть перекрывающиеся подзадачи и выбор на каждом шаге.
3. Рекуррентность по одному параметру.

## Паттерны

- **Линейный DP** — `dp[i]` зависит от `dp[i-1]`, `dp[i-2]` (Climbing Stairs, House Robber).
- **DP по сумме** — `dp[s]` = можно ли набрать сумму `s` (Coin Change, Partition Equal Subset Sum).
- **LIS** — `dp[i]` = длина наибольшей возрастающей подпоследовательности, оканчивающейся в `i`.
