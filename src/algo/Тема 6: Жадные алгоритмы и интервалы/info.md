# Тема 6: Жадные алгоритмы и интервалы

> **Главная идея темы:** локально оптимальный выбор на каждом шаге даёт глобальный оптимум. Для интервалов — почти всегда сортировка по старту или концу + один проход.

---

## Когда применять

1. Можно доказать, что жадный выбор «безопасен» (не хуже любого другого).
2. Задача на интервалы: перекрытия, объединение, минимальное число точек/удалений.
3. Локальный оптимум накапливается (Kadane, Jump Game).

## Паттерны

- **Kadane's** — `cur = max(x, cur + x)`, `best = max(best, cur)`.
- **Жадный охват** — расширяем границу достижимости (Jump Game, Partition Labels).
- **Интервалы**: сортировка по `end` (или `start`) + слияние/подсчёт.
