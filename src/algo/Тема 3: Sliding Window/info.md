# Тема 3: Sliding Window

> **Главная идея темы:** окно фиксированной или переменной ширины скользит по массиву/строке. Вместо пересчёта окна с нуля — добавляем/убираем один элемент за O(1).

---

## Когда применять

1. Нужно найти **подмассив/подстроку** под условие (длина, сумма, уникальность).
2. Условие монотонно: при расширении окна свойство «ухудшается/улучшается» предсказуемо.

## Паттерны

- **Fixed window** — окно размера `k`, двигаем целиком, обновляем сумму/счётчики.
- **Variable window** — расширяем `right`, пока можно; сжимаем `left`, пока условие нарушено.
- **Частотный словарь окна** — `Map`/массив счётчиков для проверки «все символы в окне».
