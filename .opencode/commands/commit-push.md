---
description: Добавить изменения, создать коммит и запушить текущую ветку
---

Выполни Git-коммит и push безопасно:

1. Покажи краткий статус через `git status --short`.
2. Если изменений нет — сообщи об этом и остановись.
3. Добавь все изменения, включая новые и удалённые файлы: `git add -A`.
4. Создай осмысленное сообщение коммита на основе изменений. Если пользователь передал текст после команды, используй его как сообщение: `$ARGUMENTS`.
5. Выполни `git commit -m "<сообщение>"`.
6. Запушь текущую ветку. Для новой ветки используй `git push -u origin HEAD`; если upstream уже настроен — `git push`.
7. Покажи итог: хеш коммита, название ветки и результат push.

Не выполняй `git push --force`, не изменяй историю коммитов и не коммить секреты (`.env`, ключи, токены).
